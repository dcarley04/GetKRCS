# GetKRCS!

A [2048](http://gabrielecirulli.github.io/2048/) clone for **King's Ridge Christian School** (Alpharetta, GA). Swipe and merge the Tigers' local rivals to reach the almighty KRCS tile!

[Play it here!](https://dcarley04.github.io/GetKRCS/)

## The tile ladder

| Tile | School |
|------|--------|
